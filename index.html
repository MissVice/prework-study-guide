HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML provides a framework that makes it easy for users to navigate the webpage.
  • The Header element must be in order from h1 to h6. h1 should only appear once per page.

CSS

  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.
  • rel specifies the relationship between the document and the linked document resource.
  • External style sheet is the most common way to style a webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all files for pushing
  • git commit -m "add message": leaves a message that describes what is being commited in the files
